### Getting Started

1. Clone the repository
```bash
git clone https://github.com/AdrianGroszek/smokeless-app.git
cd smokeless-app
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the app
```bash
npx expo start
```
